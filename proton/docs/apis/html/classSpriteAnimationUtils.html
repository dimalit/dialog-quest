Starts a sprite animation called animationName on pEntity.

The duration of the animation is set by animationDuration in milliseconds. It's expected that pEntity has been prepared with SetupSpriteAnimationEntity() before calling this function.

If delayToStartMS is greater than 0 then the animation is only started after this many milliseconds has passed.

When the animation ends by default the animation stops and the last frame of the animation is left visible. This behaviour can be changed by setting the finishType. For example the animation can be set to repeat itself from the start.
