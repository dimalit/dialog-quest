uint32 ColorCombine ( uint32  c1,
uint32  c2,
float  alphaMod = 1.0f 
)

Performs a "linear burn" operation for the colors c1 and c2.

This is achieved by summing the color component values together and subtracting 255 from the result. The result of this is then clamped to the accepted range of [0, 255]. Note that this operation is also performed for the alpha channel.

Finally the alpha value is multiplied with alphaMod.

http://en.wikipedia.org/wiki/Blend_modes#Dodge_and_burn

Returns:
The final resulting color of the operation.
