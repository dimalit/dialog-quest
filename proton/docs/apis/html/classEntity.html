EntityComponent * Entity::GetComponentByName ( string const &  name,
bool  bAlsoCheckParents = false 
)

Searches for an EntityComponent with the given name and returns it.

The EntityComponent is searched from this Entity's components.

If bAlsoCheckParents is true and the named EntityComponent is not found from this Entity then the parent Entity is searched also. The search continues up the Entity tree until the named EntityComponent is found or the top of the tree is reached.

If no such EntityComponent is found NULL is returned.
