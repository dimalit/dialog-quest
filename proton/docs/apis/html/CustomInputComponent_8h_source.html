00015 /**
00016  * A component that listens to specific keyboard input.
00017  *
00018  * The name of the component is initially set to "CustomInput".
00019  *
00020  * When this component is attached to an \c Entity it listens to keyboard
00021  * input and sends signals when specified keys are pressed and released.
00022  * When a key is pressed the "OnActivated" signal is sent. When a key is
00023  * released the "OnReleased" signal is sent. The variant list passed as a
00024  * parameter for both of these signals is empty.
00025  *
00026  * The following named variants are used inside the component itself:
00027  * - <b>"keys" (string):</b> specifies a set of characters that are used to
00028  *   filter the pressed/released keys. If the pressed/released key is found from
00029  *   this value then the key is accepted and the corresponding signal is sent.
00030  *   The default value is an empty string.
00031  * - <b>"keycode" (uint32):</b> specifies a single keycode that is used to
00032  *   filter the pressed/released keys. If the pressed/releasesd key's code is
00033  *   the same as this value then the key is accepted and the corresponding signal
00034  *   is sent. The default value is 0 (which is not a valid keycode).
00035  * - <b>"disabled" (uint32):</b> used to disable the component. 0 (the default) means
00036  *   the component is enabled, any other values disable it.
00037  *
00038  * The pressed/released keys are filtered with the "keys" and "keycode" variants. If
00039  * the pressed/released key is found in either of these variants the corresponding
00040  * signal is sent. If the pressed/released key is not in these variants it is silently
00041  * ignored. However if both of these variants are at their default values then
00042  * no filtering happens and all key presses/releases are signaled.
00043  *
00044  * \note This component attaches to the "OnInput" and "OnRawInput" functions of the
00045  * parent \c Entity. \c FocusInputComponent is a component that offers such functions.
00046  * You may want to use these two components together.
00047  * \see FocusInputComponent
00048  */
