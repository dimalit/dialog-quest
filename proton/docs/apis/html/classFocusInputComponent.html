Detailed Description

A component that reacts to keyboard input and passes the events to a tree of Entities.

The name of the component is initially set to "FocusInput".

This component connects to the BaseApp::m_sig_input signal and calls the "OnInput" function of the parent Entity recursively.

The following named variants are used inside the component itself:

  • "mode" (uint32): this holds a value from the eMode enum. If mode is MODE_START_NORMAL (the default) then the BaseApp::m_sig_input signal is automatically connected when this component is attached to an Entity. If mode is MODE_START_NONE then the signal is not connected.

FocusInputComponent can also listen to two other signals from BaseApp but neither of these are automatically connected.

  • BaseApp::m_sig_input_move signal can be connected by calling LinkMoveMessages(). This signal causes the "OnInput" function of the parent Entity to be called recursively.
  • BaseApp::m_sig_raw_keyboard signal can be connected by calling LinkRawMessages(). This signal causes the "OnRawInput" function of the parent Entity to be called recursively.

For all the functions of the parent Entity that this component calls the same parameters are passed that the original signal from BaseApp sends.
