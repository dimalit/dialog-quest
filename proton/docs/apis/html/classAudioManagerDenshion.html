Plays an audio specified by a file name.

Returns the audio handle for the playing sound if the playing started successfully. Returns AUDIO_HANDLE_BLANK if the playing of the audio didn't start successfully. Also returns AUDIO_HANDLE_BLANK if requesting to play a sound file and sound is currently disabled or requesting to play music and music is currently disabled.

Parameters:
fNamethe file name of the audio to play.
bLoopingwhether the sound should start again when it ends, i.e. loop.
bIsMusicspecifies if this sound is the music track.
bAddBasePathadds the base path to the file name. See GetBaseAppPath().
bForceStreamingstreams the audio directly from the file and doesn't load it into memory. This flag might not be supported by all audio backends.
Returns:
The audio handle for the playing sound if the playing started successfully. Returns AUDIO_HANDLE_BLANK if the playing of the audio didn't start.

Enables or disables the playing of music.

When the music system is disabled any currently playing music is stopped. When the music system is later enabled again the music file that was left playing is resumed from the location where it stopped.
