Variant & Variant::operator+= ( const Variant &  rhs)

Add-and-assign operator.

Adds the value of rhs to the value of this Variant and assigns the new value to this. The type of this must match the type of rhs.

The Variant type's specific addition operator is used. For example the CL_Vec2f::operator+() is used when adding two vectors together.

This operator has effect only for Variant types where adding two values makes sense and is supported, such as the float and integer types. Adding for example two pointers together makes no sense so this operator has no effect for the TYPE_ENTITY nor TYPE_COMPONENT types among others.

Note:
this operator is also supported for TYPE_STRING: it performs string catenation as expected.
Returns:
a reference to this.

bool Variant::operator== ( const Variant &  rhs) const

Equality operator.

Variants are considered equal when their types and values are equal.

If the type is TYPE_UNUSED then it doesn't matter what the values are, the Variants are considered equal.

The values are compared for equality using the values' classes' own operator==(). For example if the type is TYPE_VECTOR2 then CL_Vec2f::operator==() is used for comparing the values.
