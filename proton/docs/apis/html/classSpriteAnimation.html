const SpriteFrame * SpriteAnimation::GetFrameAtPhase ( float  phase) const

Gets the SpriteFrame in this animation at position phase.

The argument phase must be on the interval [0,1]. The phase determines a relative position on the animation. 0 signifies the beginning of the animation and 1 signifies the end.

Returns NULL if phase is invalid or this animation has no frames.
