int RandomRange ( int  rangeMin,
int  rangeMax 
)

Returns a random number between rangeMin and rangeMax.

The range includes rangeMin but not rangeMax, in other words the returned value is on the interval [rangeMin, rangeMax[.

This function is meant to be used so that rangeMin < rangeMax. However the behaviour is well defined for other combinations as well.

  • If rangeMin equals rangeMax then rangeMin is returned.
  • If rangeMin != rangeMax the returned value is on the interval [rangeMin, rangeMin+abs(rangeMax-rangeMin)[.
