void AddFocusIfNeeded ( Entity *  pEnt,
bool  bAlsoLinkMoveMessages = false,
int  delayInputMS = 0,
int  updateAndRenderDelay = 0 
)

Ensures that the given Entity has needed focus EntityComponents .

Three components will be added to pEnt, one instance of each of these classes:

  • FocusUpdateComponent
  • FocusRenderComponent
  • FocusInputComponent

If any focus component is already on the Entity they are not added again. Hence it's safe to call this method multiple times for an Entity.

Parameters:
pEntthis is the Entity where the components are added. Mustn't be NULL.
bAlsoLinkMoveMessagesif true, calls the LinkMoveMessages() method for the added FocusInputComponent.
delayInputMSspecifies a delay in milliseconds after which the FocusInputComponent is added to the Entity.
updateAndRenderDelayspecifies a delay in milliseconds after which the FocusUpdateComponent and FocusRenderComponent are added to the Entity.

Creates a new Entity and adds an OverlayRenderComponent to it.

If pParentEnt is not NULL the new Entity is added as a child to that Entity. If pParentEnt is NULL then the new Entity will have no parent.

The argument name sets the name of the new Entity.

Arguments x and y specify the position of the new Entity. The position of the OverlayRenderComponent follows the position of its parent Entity so these also set the position of that.

See also:
OverlayRenderComponent
Returns:
The newly created Entity or NULL if the creation failed for any reason.

Sets the scale of pEnt such that the resulting size of it will be vDestSize.
