Detailed Description

A manager for resources.

A resource is something that can be loaded from a file, such as an image.

The ResourceManager is useful in preventing the reading of the same file over and over again. The file reading only needs to happen once and thereafter the data is accessible through this class.

The resources are identified by their filenames from which they are loaded.

Gets a resource of any class that is or is inherited from Surface.

Returns NULL if the resource is not found or it has been previously loaded into a different type of Surface (that can't be dynamically cast to the requested type).

The textureType argument sets the texture type for the Surface. This argument however only has effect if the Surface has not been previously loaded.
