00023 /**
00024  * A component that adds button behaviour to an \c Entity.
00025  *
00026  * The name of the component is initially set to "Button2D".
00027  *
00028  * This component works nicely together with \c TouchHandlerComponent which
00029  * calls appropriate functions on the parent \c Entity. This component listens
00030  * calls to those functions and behaves like a button.
00031  *
00032  * The following named variants are used inside the component itself:
00033  * - <b>"onClickAudioFile" (string):</b> the name of the audio file that is played
00034  *   when the button is clicked. By default is whatever is the current global
00035  *   default click sound. See \c SetDefaultAudioClickSound().
00036  * - <b>"disabled" (uint32):</b> used to disable the button. 0 (the default) means
00037  *   the button is enabled, any other values disables it.
00038  * - <b>"visualStyle" (uint32):</b> determines the button's visual style, i.e. what
00039  *   the button looks like when it's pressed. This needs to be one of the values of
00040  *   \link Button2DComponent::eVisualStyle <tt>eVisualStyle</tt>\endlink.
00041  *   The default value is \link Button2DComponent::STYLE_FADE_ALPHA_ON_HOVER
00042  *   <tt>STYLE_FADE_ALPHA_ON_HOVER</tt>\endlink.
00043  * - <b>"buttonStyle" (uint32):</b> determines the button's behaviour style. This
00044  *   needs to be one of the values of \link Button2DComponent::eButtonStyle <tt>eButtonStyle</tt>\endlink.
00045  *   By default is whatever is the current global default button style. See \c SetDefaultButtonStyle().
00046  * - <b>"repeatDelayMS" (uint32):</b> a delay in milliseconds during which a button is
00047  *   non-clickable after it has been clicked. Default is 250 milliseconds.
00048  *
00049  * When the button is clicked the "OnButtonSelected" function of the parent \c Entity gets called.
00050  * This function gets a \c VariantList that contains three members:
00051  * - 0: the touch point of type \c Vector2
00052  * - 1: a pointer to the \c Entity that was touched
00053  * - 2: the id of the finger, type is uint32
00054  */
