00028  *    3. Structure your main loop like this:
00029  *

00034  *               PROFILE_BEGIN("Main loop")

00048  *           }
00049  *
00050  *       PROFILE_INIT must be called once before the
00051  *       main loop starts; it sets up the profiler and
00052  *       initializes the high-resolution timer. 
00053  * 
00054  *       Inside the loop, you wrap the pieces of code 
00055  *       that you want to measure inside PROFILE_BEGIN 
00056  *       and PROFILE_END blocks, giving each of these 
00057  *       sample points a unique name. 
00058  *
00059  *       Don't forget to call PROFILE_TALLY at the end 
00060  *       of each iteration. This turns the measurements
00061  *       into a bunch of statistics.
00062  *
00063  *    4. Provide the following two functions:
00064  * 
00065  *           void profile_clear_buffer(void);

00067  *

00069  *       be used to write the profile statistics into
00070  *       some kind of text buffer or log file.
